# ai-skills

一组面向 AI 智能体的专业 Skill 集合，每个 Skill 都是一套预定义的指令、脚本与资源，可动态加载到支持 Skill 的 AI 平台（如 Cursor、Claude Projects、Claude Code 等），让 AI 在特定专业任务上具备更强的能力。

**您不需要懂编程，只需和 AI 自然语言对话即可使用。**

---

## Skill 列表

| Skill | 简介 | 详细文档 |
|-------|------|---------|
| [DICOM Doctor](./dicom-doctor/) | AI 辅助医学影像处理：支持所有 DICOM 影像转 PNG，以及胸部 CT 的 AI 辅助阅片与 PDF 报告生成 | [README](./dicom-doctor/README.md) |

---

## 如何使用 Skill

1. 进入对应 Skill 目录，找到 `SKILL.md` 文件
2. 将 `SKILL.md` 添加到支持 Skill 的 AI 平台中
3. 在对话中用自然语言描述您的需求，AI 会自动激活对应的专业模式
