# DICOM Doctor — AI 辅助医学影像阅片 Skill

> **当前版本：v2.5.0**

---

## 简介

DICOM Doctor 是一个 AI 辅助医学影像处理 Skill，提供两项核心能力：

- **DICOM 转 PNG**：支持所有影像类型（胸部CT、腹部CT、头颅MRI、腹部MRI、核磁等）
- **AI 辅助阅片**：**当前版本仅支持胸部CT**

**您不需要懂编程，也不需要安装任何软件，只需和 AI 对话即可。**

---

## 什么是 Skill？

Skill 是一组预定义的专业指令，可以让 AI 在特定任务上具备专业能力。DICOM Doctor Skill 加载后，AI 会自动按照专业的医学影像阅片流程工作，无需您手动指导每一步。

---

## 如何启用此 Skill

将 `SKILL.md` 文件添加到支持 Skill 的 AI 平台（如 Cursor、Claude Projects 等）中即可。

启用后，当您在对话中提到以下内容时，AI 会自动激活专业阅片模式：

- DICOM 文件、`.dcm` 文件、ZIP 影像包
- CT 扫描、MRI、核磁、核磁共振、胸部 CT、腹部 CT、头颅 MRI、腹部 MRI
- DICOM 转 PNG、影像转换、肺结节、医学影像、阅片、影像报告

---

## 如何使用

### 第一步：提供影像文件

根据您的使用方式，有两种方式提供文件：

- **上传文件**（网页端 / 云端 AI）：将 DICOM 文件（`.dcm`）或包含多个 DICOM 文件的 ZIP 压缩包上传给 AI
- **指定本地路径**（本地智能体，如 OpenClaw / WorkBuddy / Claude Code 等）：直接告诉 AI 文件在本地的路径，无需上传

> 如果是胸部 CT，AI 将完成全量阅片并生成 PDF 报告。如果是其他影像类型（MRI、腹部CT 等），AI 将完成 DICOM 转 PNG 转换。

### 第二步：用自然语言描述需求

直接告诉 AI 您想要什么，例如：

> "帮我分析这个胸部 CT，看看有没有肺结节。"

> "这是我的胸部 CT 检查，请帮我生成一份影像报告。"

> "帮我把这个头颅 MRI 的 DICOM 文件转成 PNG 图片。"

> "这是腹部 CT 的 dcm 文件，帮我转换一下。"

AI 会完成分析，并生成报告。

### 第三步：获取报告

AI 完成分析后，会生成一份 PDF 格式的影像检查报告，包含：

- 检查基本信息
- AI 检视统计（正常 / 异常切片数量）
- 检查所见（结节位置、大小、类型等）
- 诊断意见与随访建议
- 分级参考（Lung-RADS）
- 免责声明

> 如果是其他影像类型（MRI、腹部CT 等），则只输出 PNG 转换结果，不生成阅片报告。

## 支持的影像类型

### DICOM 转 PNG（支持所有类型）

| 影像类型 | 检测重点 |
|----------|---------|
| **胸部 CT** | 肺结节 / 肿块 / 钙化 |
| **腹部 CT** | 肝胆胰脾肾占位 |
| **头颅 MRI** | 脑实质异常信号 / 占位 / 白质病变 |
| **腹部 MRI** | 肝脏信号异常 / 胆道 / T1T2DWI 对比 |
| **通用** | 所有其他 DICOM 影像 |

### AI 辅助阅片（当前仅支持胸部 CT）

| 影像类型 | 检测重点 | 分级系统 |
|----------|---------|---------|
| **胸部 CT** | 肺结节 / 肿块 / 钙化 | Lung-RADS |

> **注意**：AI 阅片需要支持图像识别的多模态视觉模型。腹部 CT、头颅 MRI、腹部 MRI 等其他影像类型的阅片将在后续版本中支持。
